Question 2: (25 pts) Select a year between 1952 and 2007 (remember that the gapminder data set only has data in five-year increments). In the year that you chose, group all countries in the data set into quartiles (i.e. four evenly-sized groups) based on population size. Again, in just the year that you chose, compute a Pearson correlation coefficient between life exptectancy and GDP per capita for each quartile. Display your data-frame with the correlation coefficients and p-values below.

HINTS: You can break data into quartiles using the function ntile() provided by the dplyr package. You can calculate Pearson correlation coefficients and p-values using the function cor.test().

I choose the year 2007.

gapminder %>%
  filter(year == 2007) %>% # Retrieve all data for the year 2007.
  mutate(quartile = ntile(pop, 4)) %>% # Break the data into quartiles and add a "quartile" column.
  group_by(quartile) %>% # Group by the newly-created quartile variable.
  summarize(cor_coeff = cor.test(lifeExp, gdpPercap)$estimate, # Compute correlation coefficient.
            p_value = cor.test(lifeExp, gdpPercap)$p.value, # Compute p-value.
            median_pop = median(pop)) # Record median population size for each quartile.

Since all of the p-values are less than 0.05, the correlations in each quartile are statistically significant. In each quartile, life expectancy and GDP per capita are positively correlated. I conclude that as life expectancy increases, GDP per capita also increases in the year 2007. Quartile 1, containing countries with the smallest population sizes, shows the weakest correlation between life expectancy and GDP per capita (r = 0.59), while quartile 2 shows the strongest correlation (r = 0.74).

# Retrieve all data for France, Italy, and Japan.
gapminder %>%
  filter(country %in% c('France', 'Italy', 'Japan')) -> gapminder_fij

# Create a plot with life expectancy on the y-axis and years on the x-axis
ggplot(gapminder_fij, aes(x = year, y = lifeExp, color = country)) + # Color by country
  geom_point(aes(size = gdpPercap)) + # Change the size of the points based on the GDP per capita
  geom_line() # Add a line that connects the points
